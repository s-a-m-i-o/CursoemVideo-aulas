Variaveis

servem para guardar dados

Variáveis são como "caixinhas" onde você guarda informações para usar depois.
Imagine que você precisa de um lugar para armazenar um número, uma palavra ou qualquer dado que queira utilizar em diferentes partes do código. As Variaveis servem para isso.

Então, basicamente, você usa variáveis para armazenar e lembrar de valores enquanto o programa está rodando!

var nomeDaVariavel = "valor"

let nomeDaVariavel = 5
