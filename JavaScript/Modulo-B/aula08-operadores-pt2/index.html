Operadores Aritiméticos parte 2

Operadores relacionais

  • > (Maior)
  • < (Menor)
  • >= (Maior igual)
  • <= (Menor igual)
  • == (Igual)
  • === (Identico)
  • != (Diferente)

Pra toda expressão que tenha um operador relacional ligada a ela, o resultado vai ser sempre booleano (TRUE ou FALSE)

  • 5>2 → TRUE
  • 7<4 → FALSE
  • 8>=8 → TRUE
  • 9<=7 → FALSE
  • 5==5 → TRUE
  • 4!=4 → FALSE
