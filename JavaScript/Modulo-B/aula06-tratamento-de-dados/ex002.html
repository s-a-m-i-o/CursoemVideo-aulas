Tratamento de dados

var n1 = prompr("Digite um número")

var n2 = prompt("Digite outro número")

var s = n1 + n2

alert("a soma dos números é" + s)

No caso a cima, ele não vai somar os número, só ajuntar

Porque o PROMPT retorna como STRING

Para somar com JS precisamos transformar STRING em NUMBER

var n1 = number.parseInt(window.prompr("Digite um número"))

var n2 = number.parseInt(window.("Digite outro número"))

var s = n1 + n2

alert("a soma dos números é" + s)

No caso a cima o NUMBER converte a string para number

assim o resultado será a soma dos dois números adicionados

Transformando NUMBER em STRING

alert("a soma dos números é" + String(s))

Para transformar Number em String existem duas maneiras

String(n)

n.toString()

Utilizando TemplateString ou Template literal (PlaceHolders ${})

var n1 = number.parseInt(window.prompr("Digite um número"))

var n2 = number.parseInt(window.("Digite outro número"))

var s = n1 + n2

alert(`A soma de ${n1} e ${n2} é igual a ${s}`)

${} -para utilizar placeholder

Uma forma mais facil e moderna de "concatenar"
